While Statement

While loop: In a while loop, the condition is checked before the code within
the loop is executed. If the condition is false at the start, the code within the
loop will not run at all.

Do While Statement

Do-While loop: In a do-while loop, the code within the loop is executed first,
and the condition is checked afterwards. This means the code within the loop will
always run at least once, even if the condition is false at the start.
